# sales-profits-dashboard-using-excel

## Abstract

    The Sales Performance Dashboard is designed to provide a clear and interactive view of sales and profit trends.
It helps management analyze overall performance, customer distribution, and regional growth. The dashboard focuses on 
identifying top-performing products, customers, and states while tracking profit and sales trends over time.

# Process

## Data Collection:

    Sales data containing order date, customer details, product subcategories, state, sales,
and profit values was collected and cleaned.

<a href="https://github.com/chandrakala0104/sales-profits-dashboard-using-excel/blob/main/Sales%20and%20Profit%20Dashboard%20Data.xlsx ">
raw data</a>

## Data Preparation:

Duplicates were removed from customer records.

Dates were converted into year and month columns for trend analysis.

All amount columns were formatted with a $ symbol for consistency.

## Dashboard Creation:

Pivot tables and charts were created for visual analysis.

Metrics and charts were arranged logically to show total values, yearly profit, subcategory sales, and top customer performance.



# KPI’s (Key Performance Indicators)
## KPI	Description
Total Sales	Displays the total revenue generated ($).

Total Profit	Shows overall profit earned ($).

Profit by Year	Column chart showing yearly profit trends.

Sales by Subcategory	Count of sales transactions by product subcategory.

Customer Name by Year	Unique customers served each year (duplicates removed).

Sales by States	Donut chart representing regional sales distribution.

Top 5 Customers by Profit	Identifies top customers contributing maximum profit.

Sales by Month	Line chart displaying monthly sales performance.

# Pivot Table Details

    Pivot Tables were used in this project to summarize and analyze the sales dataset efficiently. They helped transform 
large amounts of raw sales data into meaningful business insights for dashboard visualization.

## Steps Followed:

## Data Preparation

Sales data containing fields like Order Date, Customer Name, State, Subcategory, Sales, and Profit was cleaned and formatted.

All monetary columns were displayed with the $ symbol for clarity.

## Pivot Table Creation

Data range was selected, and a Pivot Table was inserted using
Insert → PivotTable → From Table/Range in Excel.

The PivotTable was placed in a new worksheet for easy organization.

<a href="https://github.com/chandrakala0104/sales-profits-dashboard-using-excel/blob/main/pivote%20tables.xlsx">
pivot table</a>

# dashboard

Profit by Year: Displayed profit trends using a column chart.

Sales by Subcategory: Counted transactions by product subcategory.

Sales by State: Visualized through a donut chart to show revenue contribution.

Top 5 Customers: Filtered using PivotTable “Top 10” feature (set to Top 5).

Sales by Month: Displayed monthly revenue pattern using a line chart.

## Visualization

Pivot Charts such as Column, Bar, Line, and Donut Charts were linked to Pivot Tables for a dynamic and interactive dashboard view.

Each chart automatically updated when the Pivot Table was refreshed.
